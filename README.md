# IMDB Play Button

A Chrome extension that adds a play button to IMDB movie pages, allowing quick navigation to playimdb.com.

## Installation

1. Download the extension
2. Open Chrome and go to `chrome://extensions/`
3. Enable Developer mode
4. Click "Load unpacked"
5. Select the extension folder

## Usage

Visit any IMDB movie page, and click the "Play" button in the top-right corner to navigate to the corresponding page on playimdb.com.

## Files

- `manifest.json` - Extension manifest
- `content.js` - Injected script for IMDB pages
- `popup.html` - Popup UI
- `popup.js` - Popup logic
- `icon*.png` - Extension icons
